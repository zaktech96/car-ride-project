# Kaizen 改善

A modern, production-ready SaaS starter template for building full-stack React applications using:

1) React Router v7 (Core Web Framework)
2) Convex (Backend-as-a-Service)
3) Clerk (Authentication)
4) Resend (Emails)
5) Polar.sh (Payments)
6) Vercel / Railway (Deployment)
7) AI Chat (OpenAI)
8) Fumadocs (Documentation)
9) Bun (Package Manager)
